# Stack Overflow Developer Survey 2025 — AI Productivity Prediction

Predicting whether professional developers agree that AI agents have 
increased their productivity, using the 2025 Stack Overflow Developer Survey.

## Notebook
`ai_productivity_prediction_eda_and_preprocessing.ipynb`  
Covers target construction, cohort filtering, feature engineering, 
EDA, preprocessing, and baseline model (Logistic Regression + Random Forest).

## Dataset
49,123 respondents, 170 columns. Cohort filtered to 9,502 professional 
developers. Final model-ready matrix: 45 features.

Source: https://survey.stackoverflow.co/2025  
Kaggle: https://www.kaggle.com/datasets/your-username/stack-overflow-developer-survey-2025
